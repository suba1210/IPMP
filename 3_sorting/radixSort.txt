Time Complexity : O(D*(N+B))

D - Maximum number of digits/characters in the data to be sorted
N - Number of elements in the array to be sorted
B - Base, if numbers, then base is 10 => we have to create 10 buckets. if alphabets the base is 26 => we have to craete 26 buckets
